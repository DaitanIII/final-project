Introduction

il cervello ha la funzione di elaborare stimoli provenienti dall ambiente esterno ed internamente dal corpo è contenuto nella cavità cranica è formato da due emisferi diviso in lobi ogni lobo e coninvolto in specifiche funzioni e in stretta connessione con il cervelletto e con il midollo spinale e formato da sostanza bianca e grigia macroscopicamente e sono visibili delle circonvoluzioni la superfice esterna degli emisferi cerebrali dell'uomo e caratterizzata dalla presenza di numerosi solchi e di rilievi che la rendono notevolmente accidentata e che ne amplificano la sua estensione sono presenti anche delle scissure che lo dividono il lobi

Obtain json object from DICOM image

Starting point for the construction of the brain in plasm is obtaining the necessary information to the construction of the curves to be used in nubs function of plasm. The original brain DICOM image are the basis for deriving the curves will be saved in a json object. To construct the json object were used drawing tools and rescue present in original brain DICOM image. In particular, the tool freePol has been used for tracing of the contours of the images concerned The resulting object is saved locally in the file JSON brain

from json to point

l'oggetto json ottenuto, deve essere ulteriormente elaborato per estrarre informazioni utili per plasm. Fruttando le librerie jquery e in particolare la funzione getJSON() si estraggono le curve di interesse che verrano scalate di un fattore di 100. l'array ottenuto è composto da un insieme di array ognuno contente le coordinate delle linee tracciate con il tool freePol. Prima del rendering, per diminuire il numero di punti da elaborare, scegliamo di disegnare le curve con un numero di punti maggiore di dieci e di prendere un punto ogni tre
